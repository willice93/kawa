Pip! (pipl.com)

One of the easiest search engines for telephone numbers is Pipl. This ts the same search page
that was discussed for name searches earlier in the book. The same search field will handle a
telephone number. This number should not be entered in the standard separated format such as
555-445-8543. Instead, enter the number without dashes or space
