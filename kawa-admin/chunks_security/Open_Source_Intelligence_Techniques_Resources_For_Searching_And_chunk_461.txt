Bing offers a cached view of many websites. Searching for a domain name, such as phonelosers.org will present many results. The first result should link to the actual website. Directly next to the website name is a small green down arrow. Clicking it will present the option of “Cached page”. Clicking this link will display a previous version of the target website as collected by Bing. Figure 3.05 (second image) displays their menu option.
